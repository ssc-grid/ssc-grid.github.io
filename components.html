#属性

名称类型默认值描述
activePage
number

当前页面号

checkboxColumn
boolean
false

是否在表格的最左边一列显示复选框

columnsModel required
array<>

表格模型，表头每一列的名称和类型
可以通过hidden来隐藏列，只有当hidden===true的时候隐藏

onCellChecked
function

选择一个单元格

onEdit
function

编辑

onPagination
function

分页

onRemove
function

删除

onSelectOne
function

选择

operateColumn
boolean
false

是否在表格的最右边一列显示操作按钮

paging
boolean
false

是否显示分页

tableData required
array<>

表格填充数据
type: boolean，数据类型是 boolean literal或者是 Boolean类型 （注意和Boolean全局对象区分）
type: ref，参照的值比较特殊，是一个object:

zuzhi: {
  id: '22EA0EB9-FABA-4224-B290-5D041A1DF773',
  code: '0403',
  name: '委外部'
}
totalPage
number

页面数量

#属性

名称类型默认值描述
defaultData
object

填充表单值
时间类型比较特殊，请先转成 ISO 8601格式的字符串 之后，再传进来。

defaultData = {
  date: new Date('2017-02-14').toISOString()
}
fieldsModel required
array<>

表单中的数据

onChange
function

当控件的值发生改变的时候触发
参数1, fieldId, 也就是传入组件中fieldsModel中的id
参数2, value, 改变之后的值
参数3, opt, 可选参数，当type为string/boolean/enum等简单类型的时候，可以 通过opt.event获取Event对象。
当type为date类型的时候，可以通过opt.formattedValue获取格式化 之后的时间值。

onReset
function

当点击“重置”按钮的时候

onSubmit
function

当表单被提交的时候触发
参数1, event, Event对象
参数2. formData, 整个表单中所有控件的值，是一个JSON对象，结构和传入参数 defaultData保持一致。
