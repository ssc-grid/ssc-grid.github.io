#列类名

定义列的className，方便添加样式，比如定义列宽

在传入的参数columnsModel中添加className来定义HTML class。 会被添加到<th>上。

比如下面例子中，可以通过为.table-head-jine添加样式来设定列宽

.table-head-jine {
  width: 100px;
}

#单元格格式化

在传入的参数columnsModel中添加formatter来指定单元格的格式化方式

#属性

名称类型默认值描述
activePage
number

当前页面号

columnsModel required
array<>

表格模型，表头每一列的名称和类型
可以通过hidden来隐藏列，只有当hidden===true的时候隐藏

onCellChecked
function

选择一个单元格

onPagination
function

分页

operationColumn
object
null

每一行是否显示操作按钮列
默认的操作按钮在最右侧的列中，如果需要指定在左侧，可以通过 align参数来设置

{
  align: 'left'
}
注意：当操作列和选择列同时存在的时候，选择列会显示在操作列的左侧

operationColumnClass
elementType
'td'

自定义的操作列组件
除非指定了operationColumn参数，否则操作列不会显示出来

paging
boolean
false

是否显示分页

selectRow
object
null

是否启用行选择，复选框/单选框
默认为null，不显示

{
  mode: 'checkbox',
  onSelect: (rowIdx, rowObj, isSelected, event) => {},
  onSelectAll: (tableData, isSelected, event) => {}
}
mode，checkbox复选，radio单选
onSelect，当选择单行的时候触发，参数：

  • rowIdx
    • 行index
  • rowObj
    • 行数据
  • isSelected
    • 复选框/单选框选中状态true/false
  • event
    • Event对象
onSelectAll，当选择所有行的时候触发，参数：
  • tableData
    • 所有行的数据
  • isSelected
    • 复选框/单选框选中状态true/false
  • event
    • Event对象
tableData required
array<>

表格填充数据
type: boolean，数据类型是 boolean literal或者是 Boolean类型 （注意和Boolean全局对象区分）
type: ref，参照的值比较特殊，是一个object:

zuzhi: {
  id: '22EA0EB9-FABA-4224-B290-5D041A1DF773',
  code: '0403',
  name: '委外部'
}
totalPage
number

页面数量

#属性

名称类型默认值描述
defaultData
object

填充表单值
时间类型比较特殊，请先转成 ISO 8601格式的字符串 之后，再传进来。

defaultData = {
  date: new Date('2017-02-14').toISOString()
}
fieldsModel required
array<>

表单中的数据 比如：

[{
  type: 'string',
  id: 'formValidationEmail',
  label: '邮箱地址',
  validation: {
    type: 'email'
  }
}]
onChange
function

当控件的值发生改变的时候触发
参数1, fieldId, 也就是传入组件中fieldsModel中的id
参数2, value, 改变之后的值
参数3, opt, 可选参数，当type为string/boolean/enum等简单类型的时候，可以 通过opt.event获取Event对象。
当type为date类型的时候，可以通过opt.formattedValue获取格式化 之后的时间值。

onReset
function

当点击“重置”按钮的时候

onSubmit
function

当表单被提交的时候触发
参数1, event, Event对象
参数2. formData, 整个表单中所有控件的值，是一个JSON对象，结构和传入参数 defaultData保持一致。

#属性

名称类型默认值描述
controlId
string

form control id

inForm
boolean
false

是否没有被SSC自己的Form组件引用

label
string

form control label

onChange
function

当文本框内容被修改时候调用

placeholder
string

文本框占位字符

validation
object

校验类型，比如

{
  type: 'email',
  helpText: '请输入正确的邮箱地址'
}
type字段支持如下类型：

  • email邮件地址

  • decimal数字，比如0.1, .3, 1.1, 1.00003, 4.0

  • int整数

  • mobilePhone手机号

  • custom自定义格式
helpText是错误提示。如果不提供，则使用默认错误提示。
如果是自定义类型，则通过matchFunc参数传递校验函数

{
  type: 'custom',
  helpText: '请输入正确的XX格式',
  matchFunc: () => {}
}
当matchFunc返回值为true的时候，认为校验通过
对于自定义类型，如果不提供helpText，则默认不显示错误提示。

value
string
''

文本框中显示的值
