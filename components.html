名称类型默认值描述
activePage
number

boolean
false

是否在表格的最左边一列显示复选框

columnsModel required
array<>

onCellChecked
function

选择一个单元格

onEdit
function

编辑

onPagination
function

分页

onRemove
function

删除

onSelectOne
function

operateColumn
boolean
false

是否在表格的最右边一列显示操作按钮

boolean
false

tableData required
array<>

totalPage
number
